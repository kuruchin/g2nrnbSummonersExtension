# SummonersExtention

Расширение для **Gothic II + Новый баланс 8.0 + Union**: дополнительные навыки призывателя у **Галлахад/Taliasan**  — лимит призванных существ, улучшение Джины и запас маны.

## Кратко

- Учитель: **Галлахад** (`VLK_6027_TALIASAN`) после линейки его квестов становления придворным магом.
- Пункт в **главном** списке диалогов: **«Изучение навыков призыва»**.
- Меню доступно после флага `**TALIASANTEACHMAGIC`** (**Галлахад** стал придворным магом и гг выбрал награду «изучение магии»).
- Для входа в ветку нужна **руна призыва особого волка** (`ItRu_SumJina` или флаг `JinaWolfReadyPet`).
- Все навыки независимы: можно учить в любом порядке, если выполнены круг магии и хватает ресурсов.

## Где учить

1. Завершить квестовую линию **Галлахад** / Taliasan.
2. Получить награду **изучение магии** у короля (флаг `TALIASANTEACHMAGIC`).
3. Иметь руну Джины (см. выше).
4. Поговорить с **Галлахад** → **«Изучение навыков призыва»**.

Без руны **Галлахад проговаривает реплику** в диалоге и возвращает в главное меню.  
**Призыватель нежити** — пункт **не показывается** (`RX_IsNecroSummoner()` из NB).  
Если круг магии не подходит или всё выучено — **всплывающая подсказка** (`AI_Print`), диалог не закрывается.

## Навыки (перки)


| №   | Название в меню                                                                       | Эффект                                                              | Круг | ОО  | Золото |
| --- | ------------------------------------------------------------------------------------- | ------------------------------------------------------------------- | ---- | --- | ------ |
| 1   | Магическая энергия призывателя +25                                                    | +25 к максимальной и текущей мане                                   | 1    | 7   | 1000   |
| 2   | Изучить улучшение Джины: снятие лимита призыва (Очков обучения: 10, Цена: 1000 монет) | Призыв Джины **не учитывается** в лимите призванных существ         | 2    | 10  | 1000   |
| 3   | Увеличение максимума призванных существ I: +1                                         | +1 к лимиту призыва (`RX_SummonCountMax` и связанные переменные NB) | 3    | 15  | 2000   |
| 4   | Увеличение максимума призванных существ II: +1                                        | Ещё +1 к лимиту призыва                                             | 4    | 20  | 4000   |


**Суммарно за всю ветку:** 52 очка обучения, 8000 монет.

### 1. Магическая энергия призывателя +25

- Увеличивает `ATR_MANA_MAX` и текущую ману на **25**.
- Самый ранний навык (1-й круг).

### 2. Изучить улучшение Джины: снятие лимита призыва

- Джина может быть призвана **при полном лимите** других существ.
- Реализация: хук `GetSummonCountMax()` и `spell_logic_sumjina` — временный неучёт слота на время каста; при активной Джине лимит для остальных призывов считается на +1 (см. `SummonersExtention_Hook.d`).
- Не добавляет отдельный слот в UI — **игнорирует** ограничение для самой Джины.

### 3–4. Расширение лимита призыва I / II

- Каждый даёт **+1** к максимуму одновременных призывов.
- Патчит `RX_SummonCountMax`, `RX_GodGiftSummonMax`, `RX_SUMMONMAX` (логика NB).
- Оба перка можно выучить; порядок между I и II не важен (кроме требования по кругу).

## Состав мода


| Файл                                        | Назначение                                                   |
| ------------------------------------------- | ------------------------------------------------------------ |
| `Scripts/SummonersExtention.d`              | Логика перков, оплата, применение бонусов                    |
| `Scripts/SummonersExtention_DIA.d`          | Диалог: permanent-тема `DIA_SE_GallahadSummon`, меню навыков |
| `Scripts/SummonersExtention_Hook.d`         | Хуки лимита призыва и каста Джины                            |
| `Scripts/SummonersExtention_Quest.d`        | Заготовка под квест (пока минимальная)                       |
| `SummonersExtention/SummonersExtention.dll` | Union-плагин (синхронизация, опрос)                          |


### Деплой в игру

```powershell
Scripts\fix_and_deploy.ps1
```

Копирует `.d` в `Gothic II\system\Autorun\` (CP1251) и DLL в `system\`.

Исходники `.d` редактировать в **UTF-8** с маркерами `SE_`*; кириллица подставляется скриптами `fix_strings.ps1` / `fix_dia_strings.ps1`.

### Диалог (важно для правок)

Пункт меню — **не** хук в подменю «Создание рун» / «Постигать суть магии», а отдельная permanent-тема:

```daedalus
INSTANCE DIA_SE_GallahadSummon (C_INFO)
{
    npc = VLK_6027_TALIASAN;
    permanent = TRUE;
    description = "Изучение навыков призыва";
};
```
