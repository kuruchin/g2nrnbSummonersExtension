## Отправная точка

У призывателя NB **уже есть**:

- **G** — атака призванных по выбранной цели (`RX_SummonNpcToTarget`, `RX_IsValidTargetToSummon`).
- **Shift+G** — сброс/сбор (`RX_ResetSummons`).
- **Shift+E** — телепорт питомцев (`KEY_SUMMON_TELEPORT`, КД).
- **Shift+R** — активный/пассивный режим (`RX_SummonMode`).
- В AB есть **`SPL_SUMMON_HEAL`** и **`RX_SummonHeal`** — лечение призванных (нужно проверить в игре: доступно ли «живому» призывателю без модов).
- **`RX_ApplyBuffToSummons`** — баф на всех призванных.
- Re-attack: `RX_SetReAttackNpc` / `RX_GetReAttackNpc` / `RX_ClearReAttackNpc`.

**Нет в AB:** `RX_Mark*`, `RX_SummonMark*` — метку добычи придётся делать в SE.

## Идея игрока (сводка)

1. Расширить роль **командира стаи**, не только «больше слотов».
2. **Лечение** суммонов (скилл или руна).
3. **Метка** на врага: +% урона от призванных, стая ускоряется и бьёт цель (связка с **G** или авто после метки).

## MVP «Метка стаи» (минимум для прототипа)

**Скрипт (концепт):**

```daedalus
var int SE_MarkVictim;      // handle/id NPC
var int SE_MarkExpireTime;  // Game_Time или счётчик
var int SE_MarkCooldown;

func void SE_CastPackMark() { ... }  // руна / активный скилл
func void SE_OnMarkTick() { ... }    // сброс метки по времени
```

**При касте:**

1. `focus` = цель под прицелом; проверка `RX_IsValidTargetToSummon` или враждебность.
2. `SE_MarkVictim = focus`; таймер.
3. `RX_SummonNpcToTarget(SE_MarkVictim)` (или аналог с параметрами из AB).
4. Жертве: временно снизить `protection[PROT_GENERAL]` **или** хук урона (если найдём функцию в AB).

**Не делать в MVP:** отдельная клавиша, мультиплеер, иконка на миникарте.

## Риски и ограничения

| Риск | Митигация |
|------|-----------|
| Логика команд в C++, не в `.d` | Хукать `RX_*` с `_Old()`, не клавиши напрямую |
| NB перезаписывает protection | C++ `SE_ReapplyPersistCap`-подобный тик или хук урона |
| Дублирование некроманта | Только `!RX_IsNecroSummoner()` у Галлахада |
| Хил уже есть в ванили NB | Перк = усиление, не «открытие пустышки» |
